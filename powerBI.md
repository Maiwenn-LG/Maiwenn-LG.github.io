## Player Performance Dashboard 🏀

**Tools** : Power BI, DAX, Excel 

**Objective:** Develop a player performance evaluation dashboard for a basketball youth academy using a custom performance rating model.

![Player Performance Dashboard](../assets/images/player-performance-dashboard.jpg)

### Project Context

This dashboard was developed for the youth academy of a French professional women's basketball club.

The objective was to create a player evaluation tool capable of assessing individual match performance over a selected period while accounting for contextual factors such as playing time, opponent strength, player age, position-specific expectations, and recent progression.

To protect privacy, all player names displayed in this dashboard have been replaced with fictional names.

### Key Features

The dashboard enables coaches and analysts to:

* Evaluate player performance over any selected period
* Compare players against position-specific benchmarks
* Analyze shooting efficiency and key performance indicators
* Track player progression over time
* Monitor an overall Performance Rating (0–35)
* Explore individual game statistics through interactive filtering

### Methodology

The Performance Rating is based on a custom evaluation model combining:

* Statistical performance
* Playing time adjustments
* Opponent difficulty adjustments
* Position-specific expectations
* Age-adjusted development targets
* Recent performance progression

The resulting score provides a standardized assessment of match performance on a 35-point scale.
