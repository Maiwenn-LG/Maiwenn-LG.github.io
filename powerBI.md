## Player Performance Dashboard 🏀

**Tools** : Power BI, DAX, Excel 

**Objective**:  ...

![Player Performance Dashboard](../assets/images/player-performance-dashboard.jpg)

## Overview

This dashboard was developed to monitor and evaluate basketball player performance over a selected period.

The project combines traditional basketball metrics (points, assists, turnovers and shooting percentages) with a custom **Performance Rating (0–35)** designed to provide a comprehensive assessment of player contributions.

## Key Features

* Interactive player selection
* Dynamic date filtering
* Position classification (Guard / Forward / Center)
* Shooting efficiency analysis
* Custom Performance Rating (0–35)
* Performance trends over time
